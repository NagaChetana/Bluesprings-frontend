# Features Page


## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

This page displays 8 feature cards:
- Check Order Status
- Track Order
- Check Stock
- Initialize Resend
- Initialize Refund
- Handle Mails
- Handle Queries
- Dashboard

Each card includes an icon, title, description, and a "see it in action" link.
